[Blar.tk](https://blar.tk) is a very simple URL shortener. It is entirely cloud-based, using Firebase Hosting for the website hosting, Firebase Cloud Firestore to store the URLs and configuration, and Firebase Functions for shortening and retrieving URLs. The source for the site is hosted [on Github](https://github.com/realprogrammer5000/blar/).
