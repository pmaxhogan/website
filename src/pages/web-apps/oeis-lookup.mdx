[OEIS Lookup](https://oeis-lookup.web.app/) is a simple website designed to help you find [OEIS](https://oeis.org/) integer sequences by providing a JS function or selecting some numbers. The source for the site is hosted [on Github](https://github.com/realprogrammer5000/oeis-lookup/).
