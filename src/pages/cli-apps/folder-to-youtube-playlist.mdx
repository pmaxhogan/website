[Folder to YouTube Playlist](https://github.com/realprogrammer5000/folder-to-youtube-playlist) is a CLI app that uploads a folder of videos downloaded with `youtube-dl` (with info JSON written) to a YouTube playlist using Google's YouTube OAuth API.
