Rogauracore GUI is a simple GUI for [wroberts](https://github.com/wroberts)'s [rogauracore](https://github.com/wroberts/rogauracore) on Linux. It functions via a taskbar menu using Electron. It requires rogauracore to be installed and in your PATH. Download the [latest release](https://github.com/pmaxhogan/rogauracore-gui/releases/latest) from [the Github](https://github.com/pmaxhogan/rogauracore-gui).
