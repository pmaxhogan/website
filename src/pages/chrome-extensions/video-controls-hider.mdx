[Video Controls Hider for YouTube™](https://chrome.google.com/webstore/detail/video-controls-hider-for/mjldjdmochmoihnmpgeboiikjinfjgco) is a simple Chrome extension that injects a simple css file into any YouTube page, hiding the video playback controls and animations when the video is fullscreen. The source code is hosted [on Github](https://github.com/realprogrammer5000/video-controls-hider).
